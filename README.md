# RAVAGE ENGINE *⚙⚙️*️

---
RAVAGE ENGINE is a lightweight engine using raylib which is built for speed and efficiency.
It excells in performance and keeps games running smoothly.

Indirectly started in March of 2025, RAVAGE ENGINE was formally started in September of 2025.
It's first game was Pixapirate, a pirate rpg test that would stress the engine 
using ai pathfinding and dynamic flanking. Now, this engine is used to make Ninja Game.

RAVAGE ENGINE is written from the ground up in c++ and uses Raylib to draw sprites.
It has full support for collsions, chunking, animations, event managment, dynamic
input mapping, and dynamic draw layers (depth sorting).

This game was created to stress RAVAGE ENGINE with heavy effects and particle heavy
combat.
